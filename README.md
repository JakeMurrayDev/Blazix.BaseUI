# Blazix.BaseUI

Blazix.BaseUI is an unofficial Blazor port of the Base UI headless component library.

## Status

Early development. The main library targets .NET 10.

## NuGet

NuGet link: TBD

## Projects

- `src/Blazix.BaseUI`: component library.
- `src/Blazix.BaseUI.Analyzers`: Roslyn analyzers used by the library.
- `demo/Blazix.BaseUI.Demo`: demo app.
- `tests`: unit and Playwright tests.

## Build

```bash
dotnet build Blazix.BaseUI.slnx
```
