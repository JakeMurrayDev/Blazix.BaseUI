# Collapsible Parity Matrix

Date: 2026-06-26

| React source surface | Required behavior | Blazor equivalent | Verification |
| --- | --- | --- | --- |
| `Collapsible.Root` | Renders default `div`, supports `open`, `defaultOpen`, `disabled`, `onOpenChange`, state callbacks, render override. | `CollapsibleRoot.razor` with `RenderElement<CollapsibleRootState>`, controlled/uncontrolled state, cancellation, `ClassValue`, `StyleValue`, `Render`. | Collapsible bUnit tests, source docs comparison. |
| `useCollapsibleRoot(open, true, true)` | Initial open reports idle; open reports starting; close reports ending; child context receives transition status. | Root initializes idle for default open, sets starting/ending on open changes, panel JS reports idle after completion. | bUnit transition state tests and Playwright animation tests. |
| `Collapsible.Trigger` | Renders button, forwards `id`, sets `type`, `aria-expanded`, open-only `aria-controls`, `data-panel-open`, disabled state, transition attributes. | `CollapsibleTrigger.razor` forwards attributes through `RenderElement`, resolves disabled from trigger/root, applies button accessibility utilities, and forwards click details. | bUnit id/attribute tests and Playwright exact `aria-controls` tests. |
| `useButton({ focusableWhenDisabled: true })` | Disabled trigger remains focusable, suppresses click/keyboard activation, does not call open-change callback. | Trigger returns before root handler when disabled; button JS handles keyboard/default behavior. | Disabled bUnit and Playwright click/keyboard tests. |
| `Collapsible.Panel` | Renders only when `keepMounted || hiddenUntilFound || mounted || open`. | `IsPresent => Context is not null && (KeepMounted || HiddenUntilFound || isMounted || CurrentOpen)`. | bUnit mount tests and Playwright DOM-state tests. |
| `hiddenUntilFound` | Closed panel remains in DOM with `hidden="until-found"` and can open from browser `beforematch`. | Panel renders `hidden="until-found"` and JS registers `beforematch`. | bUnit hidden attribute test and Playwright `beforematch` tests. |
| `beforematch` accepted | Opens with reason `none`; suppresses author motion once; restores temporary duration before close. | Root creates `CollapsibleOpenChangeEventArgs(true, None)`, context marks reason, panel passes `beforeMatchOpen` to JS, JS applies temporary `0s`, close restores it. | `AcceptedBeforeMatch_ShouldTemporarilySuppressOpenAndRestoreBeforeClose`. |
| `beforematch` canceled | Calls callback but does not open and does not arm instant-open suppression. | Root returns `false` when args are canceled; JS clears `isBeforeMatch`. | `CanceledBeforeMatch_ShouldNotSuppressNextTriggerOpen`. |
| Pending `beforematch` with trigger open | Trigger open must not wait for pending beforematch and must not inherit `0s` suppression, even if beforematch later accepts. | JS tracks `openedWhileBeforeMatchPending` and no longer awaits the pending promise in generic `open()`. Root returns `false` for superseded beforematch. | `PendingAcceptedBeforeMatch_ShouldNotSuppressInterveningTriggerOpen` and canceled variant. |
| CSS transition open | Measures expanded size, sets `data-starting-style`, then removes it after a frame. | JS `open()` measures dimensions, sets CSS vars, applies `data-starting-style`, then transitions to idle. | Playwright starting-style and CSS variable tests. |
| CSS transition close | Restores temporary beforematch style, measures current size, waits a frame, sets `data-ending-style`, waits for animations. | JS `close()` restores pending temporary styles, measures dimensions, waits `requestAnimationFrameAsync`, applies ending style, and completes after `waitForAnimationsToFinish`. | Playwright ending-style, close-height, no-flash, and hidden cleanup tests. |
| CSS keyframe initial open | Suppresses first keyframe mount animation, clears suppression when close is requested. | Panel appends `animation-name: none` while initially open and clears `shouldPreventMountAnimation` on close request; JS clears initial animation flags on close start. | bUnit initial-open suppression/reopen tests. |
| Hidden-until-found closed styles | Closed hidden-until-found transition panels persist `data-starting-style` across renders when not using CSS keyframes. | JS reports animation type; panel Razor emits persistent `data-starting-style` while hidden and non-keyframe. | bUnit persisted starting-style test and Playwright hidden cleanup test. |
| `useAnimationsFinished` one-frame close wait | Close completion waits one animation frame before reading `getAnimations()`. | Shared JS `waitForAnimationsToFinish` and Collapsible close path wait before completion detection. | Playwright close transition tests. |
| Dynamic state callbacks | Root, trigger, and panel class/style callbacks update when state changes. | `RenderElement` receives current state for each component. | `StateCallbacksUpdateAcrossRootTriggerAndPanel`. |
| Docs API surface | Docs list Root, Trigger, Panel, props, and state data attributes. | Blazor page renders equivalent component states and attributes. | `collapsible-inapp-browser-comparison.json`. |
